# MGen
Program yang dibuat untuk menmpilkan suatu model objek menjadi ASCII


-----------------------------------------------------------------------
# Instalasi
- Instal di LINUX
1. Buka terminal
2. "git clone https://github.com/Fasuyaaa/MGen.git"
3. "cd MGen"
4. "python3 MGen.py"

- Instal di WINDOWS
1. Download file zip
2. Extract
3. Jalankan di python shell
